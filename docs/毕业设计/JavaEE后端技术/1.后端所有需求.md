# 登录注册

# 挂号
一个患者可以选择某一个科室的医生挂号
需要患者id，医生id

1. 患者挂号接口，save接口
2. 根据患者id查询自己的挂号记录接口 
3. 根据医生id查询挂在自己门下的记录接口 级联 用户信息表 

# 医生

1. 根据 科室名称 查找该科室下的所有医生

# 病历

1. 根据患者编号查找对应的病历
2. 根据患者编号新建病历
# 病历节点
